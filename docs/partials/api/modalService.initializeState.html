initializeState
service in module modalService

Description

Creates a state object, populates permissions for current user, and sets display mode

Usage

initializeState(entity, [options]);

Parameters

ParamTypeDetails
entityobject

JavaScript object representing the SharePoint list item.

[options]object

Optional state params.

Returns

object

Returns the extended state.

Example

$scope.state = modalService.initializeState(request, {
    dateExceedsBoundary: false,
    enableApproval: false
});
