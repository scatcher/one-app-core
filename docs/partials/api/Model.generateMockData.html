generateMockData
service in module Model

Description

Generates 'n' mock records for testing using the field types defined in the model to provide something to visualize.

Usage

generateMockData([options]);

Parameters

ParamTypeDetails
[options]object

Object containing optional parameters.

options.quantitynumber

The requested number of mock records to return.

(default: 10)

options.permissionLevelstring

Sets the mask on the mock records to simulate desired permission level.

(default: FullMask)

options.staticValueboolean

By default all mock data is dynamically created but if set, this will cause static data to be used instead.

(default: false)
