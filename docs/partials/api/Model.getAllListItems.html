getAllListItems
service in module modelFactoryModel

Description

Inherited from Model constructor Gets all list items in the current list, processes the xml, and caches the data in model.

Usage

getAllListItems();

Returns

object

Promise returning all list items when resolved.

Example

//Taken from a fictitious projectsModel.js
    projectModel.getAllListItems().then(function(entities) {
        //Do something with all of the returned entities
        $scope.projects = entities;
    };
