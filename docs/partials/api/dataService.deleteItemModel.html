deleteItemModel
service in module dataService

Description

Typically called directly from a list item, removes the list item from SharePoint and the local cache.

Usage

deleteItemModel(model, entity, [options]);

Parameters

ParamTypeDetails
modelobject

Reference to the entities model.

entityobject

JavaScript object representing the SharePoint list item.

[options]object

Optional configuration params.

options.targetArray

Optional location to search through and remove the local cached copy.

(default: item.getContainer())

options.updateAllCachesboolean

Search through the cache for each query on the model to ensure entity is removed everywhere. This is more process intensive so by default we only delete the cached entity in the cache where this entity is currently stored.

(default: false)

Returns

object

Promise which resolves when the operation is complete. Nothing of importance is returned.
