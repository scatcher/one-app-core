updateAllCaches
service in module dataService

Description

Propagates a change to all duplicate entities in all cached queries within a given model.

Usage

updateAllCaches(model, entity, [exemptQuery]);

Parameters

ParamTypeDetails
modelobject

Reference to the entities model.

entityobject

JavaScript object representing the SharePoint list item.

[exemptQuery]object
  • The query containing the updated item is automatically updated so we don't need to process it.

Returns

number

The number of queries where the entity was found and updated.
