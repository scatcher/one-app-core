saveChanges
service in module ListItem

Description

Updates record directly from the object

Usage

saveChanges([options]);

Parameters

ParamTypeDetails
[options]object

Optionally pass params to the data service.

options.updateAllCachesboolean

Search through the cache for each query to ensure entity is updated everywhere. This is more process intensive so by default we only update the cached entity in the cache where this entity is currently stored.

(default: false)

Returns

object

Promise which resolved with the updated list item from the server.
