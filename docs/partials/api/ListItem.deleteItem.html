deleteItem
service in module ListItem

Description

Deletes record directly from the object and removes record from user cache.

Usage

deleteItem([options]);

Parameters

ParamTypeDetails
[options]object

Optionally pass params to the dataService.

options.updateAllCachesboolean

Iterate over each of the query cache's and ensure the entity is removed everywhere. This is more process intensive so by default we only remove the cached entity in the cache where this entity is currently stored.

(default: false)

Returns

object

Promise which really only lets us know the request is complete.
