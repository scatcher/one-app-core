removeEntityFromLocalCache
service in module dataService

Description

Searches for an entity based on list item ID and removes it from the cached array if it exists.

Usage

removeEntityFromLocalCache(entityArray, entityId);

Parameters

ParamTypeDetails
entityArrayArray

Cached array of list items for a query.

entityIdNumber

The ID to evaluate against to determine if there is a match.

Returns

boolean

Returns true if a list item was successfully found and removed.
