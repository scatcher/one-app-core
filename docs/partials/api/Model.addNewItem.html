addNewItem
service in module Model

Description

Using the definition of a list stored in a model, create a new list item in SharePoint.

Usage

addNewItem(entity, [options]);

Parameters

ParamTypeDetails
entityobject

An object that will be converted into key/value pairs based on the field definitions defined in the model.

[options]object
  • Pass additional options to the data service.

Returns

object

A promise which when resolved will returned the newly created list item from there server. This allows us to update the view with a valid new object that contains a unique list item id.

           //The local query cache is automatically updated but any other dependent logic can go here
   };
